# uPiper 音声品質改善レポート - 2026年3月版

## 最優先: ノイズ除去・音声強化（ポストプロセッシング）
- Sidonモデル: 15-25dB雑音低減、PESQ 4.0以上維持
- Resemble Enhance: 5分音声を15-30秒で処理
- **既存モデルを変更せずに適用可能**
- **難易度**: 低-中 / **インパクト**: 高

## 音声品質の客観評価
- PESQ + STOI + MCDの複合評価パイプライン
- MUSHRA評価フレームワークで複数モデル比較
- **難易度**: 中 / **インパクト**: 高

## SSML対応
- `<speak>`, `<prosody>`, `<break>`, `<emphasis>` タグ
- Google/Azure/Amazon互換
- **難易度**: 中 / **インパクト**: 高

## 話速・ピッチ制御
- Variance Adaptors（FastSpeech 2方式）で持続時間・ピッチ・エネルギー制御
- SSML `<prosody rate="x" pitch="y">` タグ連携
- **難易度**: 中 / **インパクト**: 高

## 日本語アクセント精度向上
- OJAD連携でアクセントベクトル制御強化
- Accent Vector技術で強度スケーリング・複数アクセント合成
- **難易度**: 中-高 / **インパクト**: 高

## サンプリングレート向上（22050Hz→48000Hz）
- FC-HiFi-GAN / iSTFTNetによるニューラル超解像度
- **難易度**: 高 / **インパクト**: 中-高

## 感情表現
- E3-VITS: 参照音声/テキスト記述ベースで感情合成
- Emo-VITS: マルチヘッドクロスアテンション
- ECE-TTS: VADベースのゼロショット感情制御
- **難易度**: 高 / **インパクト**: 高

## Voice Cloning
- GPT-SoVITS: 1分間の音声データで高品質TTS訓練
- ZSE-VITS: ゼロショットExpressiveクローニング
- **難易度**: 高 / **インパクト**: 中-高
